# 🏏 IPL Data Analysis – Python Project

## 📌 Project Overview

This project performs an exploratory data analysis (EDA) of **Indian Premier League (IPL) match data** to uncover patterns in team performance, match outcomes, and player achievements.

Using **Python data analysis libraries**, the project analyzes historical IPL match records to understand trends such as:

* Teams with the most wins
* Toss decisions and their impact on match outcomes
* Top-performing players
* Venue performance
* Seasonal trends in IPL matches

The analysis helps provide insights into how different factors influence match results in one of the world's most popular cricket leagues.

---

## 📊 Data Source

The dataset consists of **historical IPL match data**, including information about:

* Match details
* Teams
* Venues
* Toss decisions
* Match winners
* Player of the match awards

### Key Columns in the Dataset

* `id` – Match identifier
* `season` – IPL season year
* `city` – Match location
* `date` – Match date
* `team1` – First competing team
* `team2` – Second competing team
* `toss_winner` – Team that won the toss
* `toss_decision` – Decision after winning toss (bat/field)
* `winner` – Match winner
* `player_of_match` – Best performing player in the match
* `venue` – Stadium where the match was played

## 🔍 Analysis Performed

### 1️⃣ Team Performance Analysis

* Identified teams with the **highest number of match wins**
* Compared overall team performance across seasons

---

### 2️⃣ Toss Decision Analysis

Analyzed how toss outcomes affect match results:

* Toss winner vs match winner
* Impact of choosing **bat or field**

This helps understand whether toss decisions provide a competitive advantage.

---

### 3️⃣ Player Performance Analysis

* Identified players with the **most Player of the Match awards**
* Analyzed consistency and match-winning contributions

---

### 4️⃣ Venue Analysis

Examined match distribution across different stadiums:

* Venues hosting the most matches
* Venue impact on match outcomes

---

### 5️⃣ Seasonal Trends

Analyzed how IPL matches evolved across seasons:

* Number of matches per season
* Team participation and performance trends

---

## 📈 Key Insights

* Some teams consistently dominate the IPL across seasons.
* Toss decisions sometimes influence outcomes, but they are not always decisive.
* Certain players repeatedly win **Player of the Match**, indicating strong match impact.
* A few venues host a majority of IPL matches, highlighting popular cricket stadiums.
